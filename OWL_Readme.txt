
Setting up the Online Wildlife Library Website

Requirements

The OWL Website requires the following software on a Windows 7 machine:

1) Visual Studio 2015
2) Microsoft SQL Server Express 2014
3) ASP
4) IIS

The files are located in (root install location)\Document\Software Continuation and Management

Software Continuation Management Documents

There are three documents which describe setting up and testing the components of the OWL Website:

OWL-SGMD.docx- Explains how to build the C# components.

OWL-IIS-Deployment-Doc.docx- Document describes setting up the web server on IIS for connecting to the C# component.

OWL Troubleshooting.docx-  Describes a sequence to run the software for testing once the first two components are installed.
